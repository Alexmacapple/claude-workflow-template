=====================

2. Memoire workspace   : ./CLAUDE.MD                      [Workflow specifique - CE workspace]
3. Memoire projet      : ./active/projet/CLAUDE.MD        [Specs projet]
4. Memoire locale      : ./active/projet/claude.local.md  [Secrets, notes perso - GITIGNORED]

COMMANDES DISPONIBLES:
======================
/init <nom-projet>          Creer un nouveau projet avec structure complete
/verifier-regles-md         Verifier conformite Markdown (audit complet)

SCRIPTS DISPONIBLES:
====================
./check-markdown-rules.sh   Verification rapide Markdown (CLI)
./setup.sh                  Installation interactive (liens symboliques + hooks + alias)

HOOKS GIT:
==========
pre-commit                  Bloque commits avec violations Markdown

PROCHAINES ETAPES:
==================
1. Personnaliser CLAUDE.MD avec vos conventions
2. Creer votre premier projet : /init mon-premier-projet
3. Lire le guide pratique : cat MON-PREMIER-PROJET-HOW-TO.MD
4. Explorer les README.MD de chaque dossier
