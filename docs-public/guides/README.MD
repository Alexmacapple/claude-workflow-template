# Guides

Guides techniques complets et detailles servant de reference

---

## Objectif

Creer des guides exhaustifs qui servent de :
- Documentation de reference
- Resource a consulter regulierement
- Compilation de bonnes pratiques
- Guide complet sur un sujet

---

## Difference avec Articles et Tutoriels

### Article
- Opinion ou experience ponctuelle
- 1000-3000 mots
- Se lit d'une traite

### Tutoriel
- Apprentissage par la pratique
- Etapes a suivre
- Projet concret

### Guide
- Documentation complete
- Reference a consulter
- Exhaustif et maintenu
- Structure en chapitres

## Structure d'un Guide
